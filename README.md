# Vitalys 🏋️

Application web de nutrition et de sport développée avec Flask (Python).

## Fonctionnalités
- Calcul de l'IMC et recommandations personnalisées
- Plans alimentaires adaptés à l'objectif (perte de poids, prise de masse, maintien)
- Programmes sportifs (gym, pilates, yoga)
- Journal de progression (repas, poids, pas)
- Annuaire de salles de sport en France

## Technologies utilisées
- Python / Flask
- SQLite / SQLAlchemy
- HTML / CSS / JavaScript
- Déployé sur Render

## Lancer en local
```bash
pip install -r requirements.txt
python app.py
